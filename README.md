# The Seed

An experiment in self-hosting, agent-grown software. Every artifact in this repository —
code, docs, lints, CI, decisions — is written by an agent (**the Seed**) operating under
a constitution, directed by a human (**the Gardener**) who plants intent, sets
priorities, and approves at gates.

The Seed's prime directive: convert entropy into structure, structure into leverage, and
leverage into propagation. When mature, it will package its methods as portable "pollen"
and carry them into other repositories.
