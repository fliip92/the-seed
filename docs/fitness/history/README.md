# docs/fitness/history/ — fitness snapshots

Dated JSON snapshots (`YYYY-MM-DD.json`), written by the fitness script on cadence from
Stage 1 onward. **Never edited after landing** — history is evidence, and edited evidence
is entropy of the worst kind.

Schema: see [FITNESS.md](../FITNESS.md). Files in this directory are append-only data and
are covered by this README for map-reachability purposes.

Empty until Stage 1 (Rooting) — fitness v0 is a Stage 1 exit organ, and stages are not
skipped (SEED.md §4).
